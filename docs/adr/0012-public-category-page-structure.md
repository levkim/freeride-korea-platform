# ADR 0012: Public Category Page Structure

## Status
Accepted

## Context
The public navigation already included Athlete Program, Safety Education, Culture, and SHOP, but those categories did not yet have dedicated page structures. This made the site feel unfinished and pushed implementation too quickly toward admin and persistence work.

## Decision
Add first-version public pages for:

- Athlete Program
- Safety Education
- Culture
- SHOP

Each page should define the category purpose, expected content blocks, application or inquiry path, and future CMS direction. These pages are not final marketing pages; they are structural product pages that can later connect to admin-managed content.

## Consequences
- Public navigation now resolves to real category pages.
- Athlete and education pages establish the member submission and admin publishing model.
- Culture is intentionally editorial-first instead of a fully open forum at launch.
- SHOP is positioned as a future commerce and payment hub, not just merchandise.
