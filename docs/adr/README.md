# Architecture Decision Records

This folder stores ADRs for the FREERIDE KOREA project.

## Rule

- One important decision per ADR.
- Do not rewrite history. If a decision changes, create a new ADR and mark the old one as Superseded.
- Keep each ADR short enough to read quickly.

## Status

- Proposed
- Accepted
- Superseded
- Deprecated
- Rejected
