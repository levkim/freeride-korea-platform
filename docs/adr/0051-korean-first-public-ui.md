# ADR 0051: Korean-First Public UI

## Status

Accepted

## Context

FREERIDE KOREA는 국제 대회와 해외 프리라이드 문화를 다루지만, 현재 웹사이트의 주 이용자는 한국 사용자다. 관리자 입력 폼에서 영문 동시 입력을 제거한 뒤에도 공개 웹사이트에는 영문 제목, 영문 요약, 영문 버튼 라벨이 일부 남아 있어 화면 이해와 운영 일관성이 떨어졌다.

## Decision

공개 웹사이트와 관리자 목록의 기본 UI 문구는 한글 우선으로 표시한다. 콘텐츠 제목, 요약, 본문은 `ko` 값을 화면에 표시하고, `en` 값은 기존 데이터 구조와 향후 다국어 전환 가능성을 위해 내부 호환 필드로 유지한다.

FWT, FIS, YouTube, FREERIDE KOREA처럼 대회명, 서비스명, 브랜드명에 해당하는 고유명사는 그대로 유지한다.

## Consequences

- 한국 사용자가 관리자/웹사이트 화면을 더 쉽게 이해한다.
- 운영자는 한글 콘텐츠 작성에 집중할 수 있다.
- 향후 완전한 다국어 전환이 필요하면 기존 `en` 필드를 다시 표시 계층에 연결할 수 있다.
