# ADR 0025: Education Menu and Freeriding Course

## Status

Accepted

## Context

The public navigation originally used `Safety Education`, but the section now needs to include a broader education structure: avalanche safety, freeriding skills, backcountry development, and WFR-related mountain response.

## Decision

Rename the visible menu label from `Safety Education` to `Education` while keeping the existing `/safety-education` route for URL stability.

Add `FREERIDING 교육` as a dedicated education item between avalanche safety and backcountry development. This course covers downhill skills for skiers and snowboarders on natural snow, including line choice, speed control, turn control, terrain use, and group riding etiquette. It may be operated directly or delegated to professional instructors or external education centers.

## Consequences

The Education page now represents a broader learning category instead of only safety training. Backcountry development remains inside Education, directly below FREERIDING education. Existing links continue to work because the route path has not changed.
